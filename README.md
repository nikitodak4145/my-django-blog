# Мой Django Блог

Полноценный блог на Django с API и админкой.

## Функционал
- 📝 Статьи с CRUD операциями
- 🔍 Поиск по статьям
- 📄 Пагинация (разбивка на страницы)
- 🎯 REST API (JSON)
- ⚙️ Админка Django для управления
- 💻 JavaScript с OOP

## Скриншоты
1. Главная страница - список статей с поиском
2. Админка - управление контентом  
3. API - JSON данные статей
4. Код - views.py с логикой

## Технологии
- Python 3.x
- Django 5.x
- SQLite
- Django REST Framework
- HTML/CSS/JavaScript
